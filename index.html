HTML as One of My Core Skills

What is HTML?

HTML is a markup language used to create and structure web pages using tags.

Structural Elements

The <html> tag is the root element of the document.

The <head> contains metadata like the title.

The <body> contains all visible content.

Text Formatting

<h1> to <h6> are used for headings.

<p> is used for paragraphs.

<strong> is used to emphasize important text.

Lists

<ul> creates an unordered list.

<ol> creates an ordered list.

<li> defines list items.

Links

<a> is used to create hyperlinks for navigation.

Images

<img> is used to display images on a webpage.

Tables

<table> is used to organize data into rows and columns.

<tr> defines a table row.

<td> defines a table cell.

Forms

<form> is used to collect user input.

<input> allows users to enter data.

Semantic Elements

Tags like <header>, <nav>, <section>, and <footer> give meaning to the structure of the page.
